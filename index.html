Q&A

What is Addiction tracker?

Addiction tracker is an open source project designed to help those struggling with addictions or other unwanted habits to break free of their addictions!

Is Addiction tracker free?

Addiction tracker is and always will be free! It is also open source.

Where is my data stored?

Addiction tracker stores all data locally in the browser using Javascript. This means your data never leaves your PC. This means there is no need to log in or have an account, This keeps your data private and means our open source project doesn't need expensive servers to operate.

What does open source mean?

Open source is where the source code (what makes up the page you see and the code that runs the behind the scenes) is available for all to see. This means that developers can suggest improvement's and have them implemented making a better experience for you the user!

Why is Addiction tracker open source?

Addiction tracker is open to source to allow improvements from all the brilliant minds in web development community!

Does Addiction tracker diagnose illness or addictions?

NO. Addiction tracker does not diagnose illness or addiction, if you believe you have an addiction please consult a Doctor or Medical professional. Addiction tracker is simply a tool to help those with addictions/unwanted habits to track and overcome them.
